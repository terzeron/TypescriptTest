# Requisite
* `npm install -g ts-node`
* `npm install -g typescript`
* `npm install -g gulp-cli`
* `npm install --save-dev typescript gulp gulp-typescript`

# project setup
* `mkdir proj`
* `cd proj`
* `mkdir src dist`
* `npm init`
* `npm install --save-dev typescript gulp gulp-typescript`
